## ✨ 主要功能 (Features)

這個專案整合了三項實用的數學功能，並採用 RWD 響應式設計，手機電腦都能完美顯示：

### 1. 🔢 智能數字計算器
- 支援基礎四則運算（`+`, `-`, `*`, `/`）。
- 實時錯誤偵測，防止輸入無效格式。

### 2. 🔍 因數探索者 (Factor Finder)
- 輸入任意正整數，立即列出**所有因數**。
- 使用彈跳視窗 (Modal) 展示結果，版面整潔不雜亂。

### 3. ⚡ 質數生成器 (Prime Generator)
- 快速篩選並列出小於輸入數字的**所有質數**。
- 視覺化標籤設計，讓數據一目瞭然。
